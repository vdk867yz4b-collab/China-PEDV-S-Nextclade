# Nextclade Dataset: China PEDV S

This directory is the loadable Nextclade dataset.

Use in Nextclade Web:

```text
gh:Chengyan611/China-PEDV-S-Nextclade@main@/nextclade_dataset_pedv_s_genbank_china_isolates
```

Required dataset files:

- `pathogen.json`
- `reference.fasta`
- `genome_annotation.gff3`
- `tree.json`

Additional file:

- `sequences.fasta`: small example query file for testing dataset loading and genotype assignment.

Dataset summary:

- virus: porcine epidemic diarrhea virus;
- gene: spike(S);
- source: GenBank China PEDV S isolates;
- reference: `JN547228.1`;
- number of reference tips: 2011;
- clades: G1a,G1b,G2a,G2b,G2c,S-INDEL.
