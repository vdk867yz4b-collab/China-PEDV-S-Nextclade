# Supplementary Files

This folder contains curation and build-support files for the China PEDV S Nextclade dataset.

| File | Description |
| --- | --- |
| `metadata.tsv` | Tip metadata used in the curated reference set. |
| `clade_assignments.tsv` | Final clade assignment table used for validation. |
| `source_clade_labels.tsv` | Source clade labels used to check dataset clade calls. |
| `tree.nwk` | Newick tree corresponding to the 2011-sequence reference set. |
| `prep_summary.json` | Summary of sequence filtering and input preparation. |
| `build_summary.json` | Summary of final dataset build and validation. |

These files are provided for transparency and reproducibility. They are not required for routine Nextclade Web use.
